# Space-Track Console
A Python-based command-line interface (CLI) for managing space missions.

## Purpose
This project demonstrates foundational programming concepts including:
* User input handling and data validation.
* Control flow using `while` loops and `if-elif-else` statements.
* Functional programming structure in Python.

## How to Run
1. Ensure you have Python installed.
2. Run `python space_tracker.py` in your terminal.
